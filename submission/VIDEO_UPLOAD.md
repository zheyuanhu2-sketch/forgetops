## Description

ForgetOps is a lineage-aware privacy-operations agent that turns a right-to-erasure request into a bounded, approval-gated, verifiable workflow. DataHub schema, lineage, ownership, and governance context determine the exact scope; protected outcomes remain visible; every permitted action is verified; and approved evidence is read back from a fresh mutation-disabled MCP session.
